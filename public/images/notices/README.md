상단 공지사항 이미지는 이 폴더의 파일을 교체하거나 `src/data/site.ts`의 `topNotices` 배열에서 각 공지의 `image` 경로를 바꿔서 반영할 수 있습니다.

- 작업 원칙: 공지 이미지는 모두 `1:1 비율`
- 권장 비율: `1:1`
- 현재 연결된 파일: `1.png`, `2.png`, `3.png`
- 현재 공지 링크: 없음

공지 링크는 `src/data/site.ts`의 각 공지 객체 `href`에 입력하면 활성화됩니다.
공지사항은 `topNotices` 배열에 항목을 추가하면 계속 늘릴 수 있습니다.
