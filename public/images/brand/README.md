Put BI image assets in this folder.

Recommended filenames:
- bi-logo.png
- bi-hero.png

Current bindings:
- Header logo: `/images/brand/bi-logo.png`
- Hero image: `/images/brand/bi-hero.png`

Current live assets in this project:
- Header logo: `/images/brand/앱 아이콘.png`
- Hero image: `/images/brand/가로형 bi.png`

If you want different filenames or extensions, update `src/data/site.ts`.
